# Programação para Internet I

Repositório desenvolvido durante a Unidade Curricular **Programação para Internet I**, do Curso Superior de Tecnologia em Sistemas para Internet do Instituto Federal de Santa Catarina (IFSC).

Ao longo da disciplina foram desenvolvidas atividades práticas envolvendo HTML5, CSS3, JavaScript, Git/GitHub e consumo de APIs, culminando na construção de um projeto web completo.

---

## Objetivos da disciplina

Este repositório reúne os exercícios e projetos desenvolvidos com o objetivo de praticar:

- Estruturação de páginas com HTML5 semântico;
- Estilização utilizando CSS3;
- Layouts responsivos com Flexbox;
- Manipulação do DOM com JavaScript;
- Validação de formulários;
- Consumo de APIs utilizando Fetch;
- Versionamento de código com Git e GitHub.

---

# Projeto Final

O principal projeto desenvolvido na disciplina foi o site **Receitas de Família**, uma aplicação web voltada ao compartilhamento de receitas culinárias.

### Funcionalidades

- Página inicial com receitas em destaque;
- Layout responsivo;
- Cards com imagens;
- Navegação entre páginas;
- Formulário para envio de receitas;
- Validação de formulário com JavaScript;
- Interface amigável e organizada.
